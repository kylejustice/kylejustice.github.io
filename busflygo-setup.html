Bus-Fly-Go

Install + Run

Enclosed below are instructions to install and execute the Bus-Fly-Go program. It is assumed that you have access to the 'busflygo.tar.gz' tarball and solid internet connection.

Install Bus-Fly-Go & Setup Environment

1) Install Ubuntu 16.04 LTS

You will need to use an installation of Ubuntu Linux to run the application. Older versions might work, but this manual was confirmed to work on Ubuntu Desktop 16.04 LTS. Installing a fresh Ubuntu 16.04 on Oracle VM VirtualBox is a convenient option.

For these next steps, you will need to use the terminal command line in your Ubuntu installation. Do NOT type the '$' [or characters before it] when executing a command. The '$' only signals the start of a new command.

Copy, or type the following highlighted commands into the terminal, and follow along with the instructions.

2) Install Git

$ sudo apt-get install git

3) Install Ruby Dependencies

$ sudo apt-get install curl zlib1g-dev build-essential libssl-dev libreadline-dev libyaml-dev libsqlite3-dev sqlite3 libxml2-dev libxslt1-dev libcurl4-openssl-dev python-software-properties libffi-dev

4) Install ruby-build

$ git clone https://github.com/sstephenson/ruby-build.git ~/.rbenv/plugins/ruby-build

5) Install rbenv

$ sudo apt install rbenv

6) Install Ruby (takes a while)

$ rbenv install 2.2.2

7) Set global Ruby version

$ rbenv global 2.2.2

8) Install nodejs

$ sudo apt-get install nodejs

9) Install ruby-dev

$ sudo apt-get install ruby-dev

10) Install Rails (takes a while)

$ sudo gem install --no-document rails -v 4.2.1

11) Install redis

Install redis with these four separate commands. Also make sure to navigate back home with $ cd.

$ wget http://download.redis.io/redis-stable.tar.gz
$ tar xvzf redis-stable.tar.gz
$ cd redis-stable
$ make
$ cd

12) Install redis-server

$ sudo apt install redis-server

13) Extract busflygo.tar.gz

Download busflygo.tar.gz and place it in your home directory. From your home directory, extract the application by executing:

$ tar -xvzf busflygo.tar.gz

14) Navigate into the busflygo directory

$ cd busflygo

15) Run the bundle install

~/busflygo$ bundle install
Okay, everything should now be installed. Let's run Bus-Fly-Go!

Run Bus-Fly-Go

1) Open up three separate terminals in Ubuntu

To run Bus-Fly-Go 100%, three command terminals are required.
Navigate each of the three terminals to the busflygo folder. ($ cd busflygo x3)

2) In one terminal, startup Rails

Boot up Rails from one of the three terminals.

~/busflygo$ rails s

IMPORTANT: During the first startup of Rails, WAIT for PhantomJS to finish installing.

3) In one other terminal, startup redis-server

Boot up redis-server from one of the other three terminals.

~/busflygo$ redis-server

4) In the third terminal, startup Sidekiq

Boot up Sidekiq from the third untouched terminal.

~/busflygo$ bundle exec sidekiq

5) Open a web browser, and navigate to the application

Open up a web browser in Ubuntu, and navigate to the locally hostel web application.
The default address should be http://localhost:3000
If, for some bizarre reason, the link doesn't work, look at the command terminal running Rails. Rails will state which address the local website is running on.

Bus-Fly-Go should be fully functional and running!
